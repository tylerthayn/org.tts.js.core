@js/core
======
[![npm version]][npm]
[![license badge]][license]

TTS Design's javascript core libraries for GScript, NodeJS & Browser implementations

## Usage

#### 1. NodeJs

```npm install @js/core```

```
require('@js/core')
```

#### 2. Browser

##### CDN

```<script src="https://cdn.ttscloud.site/@js/core.js"></script>```

##### Local source

```<script src="@js/core.js"></script>```
